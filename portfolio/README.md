# Biruthivin BK — Data Analyst Portfolio

A stunning, production-ready single-page portfolio built with React, Tailwind CSS, Framer Motion, and Recharts.

## ✨ Features

- **Dark / Light Mode** — Smooth animated toggle (Sun/Moon) with full theme propagation
- **Hero Section** — Looping animated SVG line chart with glowing Electric Cyan accent
- **Bento Box Skills Grid** — Asymmetric, responsive grid with hover glow and animated progress bars
- **Experience Timeline** — Animated vertical timeline with internship/full-time distinction
- **Interactive Projects** — Cards with embedded Recharts graphs (Bar, Area, Line, Pie) that adapt to theme
- **Certifications** — Clean two-column grid layout
- **Contact Section** — Clickable links for email, phone, LinkedIn, and GitHub
- **Framer Motion animations** — Scroll-triggered reveals, stagger children, path-length SVG, spring toggles

## 🚀 Quick Start

```bash
# 1. Install dependencies
npm install

# 2. Start dev server
npm run dev

# 3. Build for production
npm run build
```

## 🛠️ Tech Stack

- **React 18** — Functional components + Hooks
- **Tailwind CSS 3** — `darkMode: 'class'` strategy
- **Framer Motion 11** — 60fps animations, SVG pathLength, spring physics
- **Recharts 2** — Responsive charts (BarChart, AreaChart, LineChart, PieChart)
- **Lucide React** — Icon library
- **Vite 5** — Lightning-fast dev/build toolchain
